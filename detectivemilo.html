Overview

The Brilliant Detective Milo is a narrative-driven mystery game developed as part of a large interdisciplinary team project at UC Irvine, bringing together departments across design, programming, art, audio, writing, and production.

The project emphasized collaborative development and iterative playtesting, resulting in a polished visual direction and smooth gameplay experience showcased during presentation.

UI/UX Leadership Interface Design Figma Adobe Illustrator Cross-Team Collaboration

Players take on the role of Detective Milo, investigating mysteries through interrogation, document inspection, and deduction. The game features a fully interactive diegetic UI — case files, suspect folders, witness testimonies, and autopsy reports are all physical objects within the game world.

Strong focus on narrative clarity and immersion meant UI had to support rather than interrupt the storytelling experience at every moment.

Responsibilities

UI/UX Leadership

Defined the overall user experience direction and ensured consistency across all interface elements. Established UI design standards and visual guidelines, managed task distribution and iteration cycles within the UI/UX team.

Interface Design

Designed and prototyped all in-game UI elements in Figma — including dialogue interfaces for Yarn Spinner narrative systems, menu systems and navigation flows, HUD elements, and interactive prompts. Visual assets produced in Adobe Illustrator.

User Experience Design

Mapped user flows for key gameplay interactions, reduced friction in navigation and dialogue, and ensured readability, clarity, and accessibility across all UI elements.

Cross-Team Collaboration

Coordinated with writers to align UI with dialogue pacing and tone, worked with programmers to ensure designs were technically feasible, and synced with artists to maintain visual consistency across the entire project.

04

Design Approach

Iterative Prototyping

All UI systems developed through rapid iteration cycles, beginning with low-fidelity wireframes and progressing to polished mockups. Playtesting feedback was continuously incorporated to refine usability and clarity at each stage.

Narrative-Driven UI

Given the game's emphasis on storytelling, UI design prioritized clarity and immersion — interfaces were designed to enhance rather than interrupt the narrative experience. Diegetic elements like case files and suspect folders blur the line between UI and world.

Scalability & Consistency

Established reusable UI components and consistent visual patterns to maintain cohesion across the project — especially critical given the 20+ member team and distributed development workflow.
